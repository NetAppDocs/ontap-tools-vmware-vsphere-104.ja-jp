= アプリケーションユーザーのパスワードをリセットする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
アプリケーション ユーザー パスワードは、vCenter Server への SRA および VASA プロバイダの登録に使用されます。

.手順
. Web ブラウザからONTAPツール マネージャを起動します。 `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. 導入時に指定したONTAP tools for VMware vSphere管理者認証情報を使用してログインします。
. サイドバーから*設定*を選択します。
. *VASA/SRA 資格情報* 画面で、*パスワードのリセット* を選択します。
. 新しいパスワードを入力し、新しいパスワードの入力を確認します。
. 変更を反映するには、[*リセット*] を選択します。
