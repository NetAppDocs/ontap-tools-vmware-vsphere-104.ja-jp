= NFSおよびVMFSデータストアのサイズ変更
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
データストアのサイズを変更すると、仮想マシン ファイル用のストレージを増やすことができます。インフラストラクチャの要件の変化に応じて、データストアのサイズを変更できます。

*このタスクについて*

NFS および VMFS データストアのサイズのみを増やすことができます。  NFS および VMFS データストアの一部であるFlexVol volumeは、既存のサイズより小さくすることはできませんが、最大 120% まで拡張できます。

.手順
. vSphere Client のホームページから、[ホストとクラスタ] を選択します。
. ナビゲーション ペインで、データストアが含まれているデータ センターを選択します。
. NFS または VMFS データストアを右クリックし、* NetApp ONTAPツール* > * データストアのサイズ変更* を選択します。
. [サイズ変更] ダイアログ ボックスで、データストアの新しいサイズを指定し、[OK] を選択します。
