= ONTAPツールマネージャのパスワードを変更する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP Tools Manager を使用して管理者パスワードを変更できます。

.手順
. Web ブラウザからONTAPツール マネージャを起動します。 `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. 導入時に指定したONTAP tools for VMware vSphere管理者認証情報を使用してログインします。
. 画面の右上隅にある*管理者*アイコンを選択し、*パスワードの変更*を選択します。
. パスワード変更ポップアップ ウィンドウで、古いパスワードと新しいパスワードの詳細を入力します。パスワード変更の制約はユーザーインターフェース画面に表示されます。
. 変更を実装するには、[*変更*] を選択します。
